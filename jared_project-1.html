Project Overview

I have always been curious about how much public opinion sways stock prices in comparison to the popular modern mathematical models accuracy. This project is my attempt to see the differences between the 2, and also get some exposure to one of the more popular trading techniques; high frequency trading. There are 3 main components to this project; sentiment based HFT, Mathematical HFT (long short term memory ML model), and a comparison between the 3.

Sentiment Based HFT

This is the first part of the project in where I will use pre-trained models through PyTorch to get sentiment score values. In my early testing I noticed that some of the more finacially literate models would strugle on sentances like "the market is looking poor" where more diverse models would struggle on sentances like "the market is quite bearish". This led me to the decision to use a handful of models where some are more general sentiment models and others are trained more on financial phrases.

For efficience, I also decided to run the models on separate threads such that not as much time would be wasted in calculations. For this, no mutexes were needed as all of the functions used were atomic as noted in the Python documentation. The general flow of this sub-project is as follows: collect data, clean data, run models on data, fit the data to a [-1,1] range, and then average the values together based on their respective weights. You can see this is the "main.py" file of the Sentiment Code Link above. This program is then ran itteratively where it will make many small simulated purchases to mimic a HFT algorithm. This part is nearly complete.

Machine Learning (Mathematical) Based HFT

This is phase 2 of the project where I will be using a popular machine learning algorithm called "Long Short Term Memory" to predict stock prices given a stock ticker. Although pre-trained models for this are readily accessible, I will be training my own model for this sub-project.

So what is a LSTM model? A Long Short-Term Memory (LSTM) model is a type of recurrent neural network (RNN) designed to handle sequential data and capture long-term dependencies. Traditional RNNs struggle with "vanishing gradients," where they fail to retain information over long sequences. LSTMs solve this by using special memory cells.

Each LSTM cell contains three gates:

  • Forget gate: Decides what information to discard from the cell state.
  • Input gate: Determines what new information to add.
  • Output gate: Chooses what information to output based on the current cell state.

The cell state, which flows through the network with minimal modifications, allows LSTMs to preserve information over longer sequences. At each time step, the gates control the flow of information, allowing the network to learn which parts of the sequence are important to remember or forget.

LSTMs are widely used for tasks like time-series prediction, natural language processing, and speech recognition because they excel at learning patterns in sequential data, where context from earlier inputs can be crucial to make accurate predictions. Making it a perfect tool to make predictions for the stock market, where each stock can easily be viewed as a time-series. This model will then be ran interatively to make purchasing decisions as an HFT. This part of the project is about to begin

Comparison

This part of the project will be a static webapp that will be hosted on Github Pages just like this website that will routinely run the prediction models over time and compare their respective scores. This project will have an indepth analysis of the differences between the two and will show their previous scores as well as their average returns. This part of the project has not begun yet
